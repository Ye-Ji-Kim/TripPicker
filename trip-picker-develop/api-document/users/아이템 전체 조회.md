## 아이템 전체 조회

| 메소드 | 경로             | 짧은 설명                     |
| ------ | ---------------- | ----------------------------- |
| GET    | /items/{userIdx} | 아이템 전체 조회(메인 페이지) |

### 요청 헤더

```
Content-Type : application/json,
```

### 요청 파라미터

| 파라미터 명 | 파라미터 값 예시 |
| ----------- | ---------------- |
| userIdx     | 1                |
