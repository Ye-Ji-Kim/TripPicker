## 회원 선택 지역 저장

| 메소드 | 경로             | 짧은 설명                   |
| ------ | ---------------- | --------------------------- |
| POST   | /users/locations | 회원 선택 지역 저장 및 변경 |

### 요청 헤더

```
Content-Type : application/json,
```

### 요청 바디

- region(지역) : 시도 + 시군구 (ex. 서울 강남구 / 강원 춘천시 / 인천 강화군)
  - 시도는 '시' , '도'를 꼭 생략한다.  

```
{
	"userIdx" : 1,
	"region" : "인천 강화군"
}
```

### 응답 바디

#### 회원 선택 지역 저장 성공

```
{
    "status": 201,
    "message": "회원 선택 지역 저장 성공",
    "data": null
}
```

#### 회원 선택 지역 저장 실패

```
{
    "status": 600,
    "message": "회원 선택 지역 저장 실패",
    "data": null
}
```
