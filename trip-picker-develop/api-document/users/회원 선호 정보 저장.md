## 회원 선호 정보 저장

| 메소드 | 경로               | 짧은 설명           |
| ------ | ------------------ | ------------------- |
| POST   | /users/preferences | 회원 선호 정보 저장 |

### 요청 헤더

```
Content-Type : application/json,
```

### 요청 바디

```
{
	"userIdx" : 1,
	"ageType" : "TWENTY",
    "addressType" : "SEOUL",
    "marriageType" : "SINGLE",
	"nightType" : "DAY",
	"monthType" : "JUL",
	"partnerType" : "COUPLE",
	"costType" : "FOURTY",
	"activityType" : "NATURE"
}
```

### 응답 바디

#### 회원 선호 정보 저장 성공

```
{
    "status": 201,
    "message": "회원 선호 정보 저장 성공",
    "data": null
}
```

#### 회원 선호 정보 저장 실패

```
{
    "status": 600,
    "message": "회원 선호 정보 저장 실패",
    "data": null
}
```
