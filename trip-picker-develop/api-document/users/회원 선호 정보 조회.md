## 회원 선호 정보 조회

| 메소드 | 경로                         | 짧은 설명           |
| ------ | ---------------------------- | ------------------- |
| GET    | /users/preferences/{userIdx} | 회원 선호 정보 조회 |

### 요청 헤더

```
Content-Type : application/json,
```

### 요청 파라미터

| 파라미터 명 | 파라미터 값 예시 |
| ----------- | ---------------- |
| userIdx     | 1                |

### 응답 바디

#### 회원 선호 정보 조회 성공

```
{
    "status": 200,
    "message": "회원 선호 정보 조회 성공",
    "data": {
        "userIdx": 1,
        "areaCode": 1,
        "sggCode": 1,
        "regionName": "서울 강남구"
    }
}
```
